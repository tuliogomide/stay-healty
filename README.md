# Welcome to your Expo app 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app (Necessary to open with MAC OS)

   ```bash
   npx expo run:ios
   ```

This aplication is made with Skia to graphics and animations, and Liquid Glass with Expo Native UI
