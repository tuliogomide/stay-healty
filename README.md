# 🥗 Stay Healty Pro:

## 📖 About the Application

This is not just another calorie counter. **Stay Healty** is a comprehensive fitness and nutrition companion designed for high-performance individuals. It moves beyond simple logging to provide a holistic view of your health based on two key pillars: **activity intensity** and **dietary structure.**

The application intelligently calculates your daily caloric needs and macronutrient distribution by analyzing:

1.  **Exercise Dynamics:** Integrating workout frequency, duration, and intensity to dynamically adjust caloric targets.
2.  **Structured Dieting:** Guiding you through flexible dieting approaches or strict meal plans, tracking protein, carbs, and fats with precision.

By leveraging a high-performance technology stack, the app provides a visceral, smooth, and highly responsive user interface that keeps you engaged with your fitness journey.

## 🚀 Technical Stack (The "How")

This project pushes the boundaries of React Native performance to deliver a premium, fluid user experience:

* **[Expo](https://expo.dev/):** Our robust foundation for universal React development and streamlined tooling.

* **[React Native Skia](https://shopify.github.io/react-native-skia/):** The core performance driver. This high-performance 2D graphics engine (by Shopify) is used for GPU-accelerated drawings, shaders, and complex animations that would choke the standard React Native bridge.

* **[Victory Native XL](https://commerce.nearform.com/open-source/victory-native/):** A modern data visualization library built entirely *on top of Skia*. We use it to render responsive, high-speed calorie and macronutrient charts.

* **[Liquid Glass]:** Used Expo Native UI component library delivering advanced glassmorphism effects, refraction, and real-time blur, creating a clean, modern aesthetic.

* **[Redux Toolkit](https://redux-toolkit.js.org/):** Our reliable, scalable, and predictable state management solution, handling everything from user profiles to complex nutrition logs.

## 🛠️ Getting Started

### Prerequisites

Ensure you have your React Native development environment set up and Node.js installed. For iOS, you need a Mac with Xcode.

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run on iOS (Requires macOS & Xcode):**
    ```bash
    npx expo run:ios
    ```
